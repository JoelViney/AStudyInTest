
Install Json Configuration
==========================
1. Install-Package Microsoft.Extensions.Configuration
2. Install-Package Microsoft.Extensions.Configuration.Json
	Now we can read config.json files.

Installing Entity Framework
===========================
1. Install-Package Microsoft.EntityFrameworkCore
	
2. Install-Package Microsoft.EntityFrameworkCore.SqlServer

3. Install-Package Microsoft.EntityFrameworkCore.Tools
	Enables these commonly used commands: Add-Migration, Drop-Database, Get-DbContext, Scaffold-DbContext, Script-Migrations, Update-Database
